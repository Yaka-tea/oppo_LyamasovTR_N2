Вариант 3

Поступление товаров

Объект и его свойства:
- дата
- название товара (строка)
- количество (целое число)

Во входной строке:
- строковые значения заключены в кавычки
- дата задаётся в формате `гггг.мм.дд`
- свойства разделяются одним или несколькими пробелами

Программа формирует программный объект на основе входной текстовой строки и выводит его свойства в консоль.

---

Формат входных данных (пример)

2023.10.05 "Гвозди" 150

---

Формат выходных данных

В консоль выводятся значения свойств созданного объекта:
- дата
- название товара
- количество

---

Стиль оформления кода

Код оформлен в стиле **Google C++ Style Guide**:
- имена типов — `CamelCase`
- имена переменных — `lower_case`
- отступы — 4 пробела
- фигурные скобки — на той же строке
- единый стиль форматирования по всему файлу

---

Используемые технологии

- Язык программирования: C++
- Стандарт: C++17
- Используемые библиотеки стандартной библиотеки:
  - `<iostream>`
  - `<string>`
  - `<sstream>`
